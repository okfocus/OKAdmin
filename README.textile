h2. Installation

p. In your Gemfile

pre. gem 'rails_admin_example_theme', :git => 'git://github.com/okfocus/OKAdmin.git'

p. Inside config/application.rb, just after Bundler.require:

pre. ENV['RAILS_ADMIN_THEME'] = 'example_theme'
